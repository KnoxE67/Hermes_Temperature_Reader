# Hermes_Temperature_Reader
Building a Custom Hermes project with a LM75AD_118, XIAO RP2040, and a 0.96” 128 X 64 OLED DISPLAY

Time spent: 2h

Worked on finding a Sensor and the Schematic by researching the JLCPCB catalog for a sensor that I thought would be cool, and settled on a Temperature sensor, which would be shown on an OLED display so that I could know the temperature in my room.
